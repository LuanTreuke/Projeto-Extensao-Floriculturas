<h3>Solução Digital para Floriculturas</h3>

📌 <strong>Descrição</strong>

Este projeto consiste no desenvolvimento de uma solução digital voltada para pequenos empreendimentos, com foco na criação de uma vitrine virtual e no gerenciamento básico de produtos e pedidos.

A aplicação permite a exposição organizada de produtos, facilitando a visualização por clientes e auxiliando na gestão interna do catálogo.

 <strong><h3>🎯 Objetivo</h3></strong>

O sistema tem como objetivo:

• Disponibilizar um catálogo digital de produtos <br>
• Permitir a visualização organizada das informações <br>
• Facilitar a comunicação entre cliente e empreendimento <br>
• Auxiliar no gerenciamento básico de pedidos <br>
<br>
<h3>🧩 Funcionalidades</h3>
• Cadastro de produtos<br>
• Edição e remoção de produtos<br>
• Listagem de produtos em catálogo<br>
• Filtro por categorias<br>
• Sistema de autenticação<br>
• Área administrativa<br>
• Gerenciamento de pedidos (estrutura preparada)<br>
<br>
<h3>🏗️ Arquitetura do Sistema</h3>

O sistema foi desenvolvido seguindo o modelo cliente-servidor:<br>

Front-end: responsável pela interface e interação com o usuário<br>
Back-end: responsável pela API, regras de negócio e persistência de dados<br>
Banco de dados: armazenamento das informações do sistema<br>
